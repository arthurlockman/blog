I stumbled across a plugin for QuickLook on the Mac today that allows you to preview Markdown files right in the QuickLook pane. It's called QLMarkdown, written by Phil Toland. Installation is a simple drag and drop. You can style the previews how you like too, just edit the styles.css file and you're all set. It works really well, and I highly recommend you install it.

Download it here.
