Today I was playing around with El Capitan's new Notes app, and was interested to see if it was possible to export notes from Evernote as HTML and ingest them into Notes. I came up with this little bash/applescript that seems to do the trick. It takes in an exported folder on the command line, converts each to a rich text file, and then pastes that into notes 1. It's not perfect, but it seems to work fairly well.

Update 2/13/2016: The next version of OS X (10.11.4) includes a feature in the Notes app that imports notes directly from Evernote. Read about it here. 

  1. Updated 6/18/2015: Added support for including attachments into created notes. It was annoying that attachments would only be included as links, and if you deleted the original folder then the links would go bad. This will add all of the note attachments as actual attachments to the created note. ↩
